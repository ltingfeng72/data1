### 🚀 下载后的使用步骤

1. **解压文件**
   - Windows: 右键 → 解压到当前文件夹
   - Linux/Mac: `tar -xzf library_system.tar.gz` 或 `unzip library_system.zip`

2. **阅读文档**
   - 先阅读 `文件清单.md` 了解项目结构
   - 再阅读 `部署指南.md` 按步骤部署
   - 遇到问题参考 `README.md`

3. **创建数据库**
   ```bash
   mysql -u root -p
   CREATE DATABASE library CHARACTER SET utf8mb4;
   USE library;
   SOURCE database/schema.sql;
   ```

4. **配置 ODBC**
   - 参考 `部署指南.md` 中的详细步骤

5. **编译运行**
   ```bash
   cd src
   g++ -o library_system main.cpp -lodbc
   ./library_system
   ```
